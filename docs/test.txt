1. **임의 문자 파형 관찰**

   * 코드로 `'a'` 또는 다른 문자를 전송하고, 오실로스코프 CH1을 D8에 연결해 파형을 관찰한다.
   * 스타트비트(LOW) → 데이터 8비트 → 스톱비트(HIGH) 순으로 나타나는지 확인한다.
   * 각 비트의 폭이 약 **104 µs(9600 bps)** 인지 측정한다.

2. **포트를 이용한 UART 모사 신호 생성**

   * `PORTB` 직접 제어를 통해 **비트뱅(bit-bang)** 방식으로 신호를 생성한다.
   * 코드에서 `TX_HIGH()` / `TX_LOW()` 명령이 출력 핀의 전압을 직접 제어하는 원리임을 설명한다.

3. **오실로스코프 2채널 동시 표시 (송신/수신 비교)**

   * CH1 = D8(송신), CH2 = D0(수신)으로 연결해 **동일 화면에 표시**한다.
   * 두 파형의 타이밍 차(지연, 왜곡)를 관찰하고 기록한다.
   * “Measure” 기능으로 각 비트의 폭, 주기, 상승·하강 시간 등을 정량 측정한다.
   * 측정값을 이론값과 비교하고 오차율(%)을 계산한다.

4. **루프백 검증 (TX→RX 점프)**

   * D8과 D0을 점퍼선으로 연결한다.
   * RX에서 받은 데이터를 다시 UART TX(USB 시리얼)로 전송해 **터미널 창에서 문자 출력 확인**.

5. **문자 반복 검증**

   * ‘a’, ‘A’, ‘Z’ 등 임의의 다른 문자로 반복 확인한다.
